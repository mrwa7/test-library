Usability: Usability refers to the ease of use and learnability of a human-made object such as a tool or device. Usability metrics include effectiveness, efficiency, and satisfaction with which users achieve specified goals in particular environments.

Model-Based Testing: Model-based testing of GUI applications involves using models to represent the desired behavior of the system under test. These models help in generating test cases that can automatically be executed to verify the functionality of the GUI applications.
